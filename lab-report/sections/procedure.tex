\section*{PROCEDURE}
\addcontentsline{toc}{section}{PROCEDURE}

\begin{enumerate}[label=\arabic*., itemsep=6pt, parsep=0pt, left=0]
\item Define the sampling parameters.
\item Generate time instances of sampling.
\item Create a sinc signal.
\item Define a cosine wave carrier signal of high frequency (must satisfy Nyquist Sampling
Theorem for defined sampling frequency).
\item Perform Modulation.
\begin{enumerate}
    \item DSB-FC
    \item DSB-SC
    \item SSB-SC
\end{enumerate}
\item Plot the modulated signals with respect to time.
\item Plot the power spectrum of the modulated signals.
\item Compare and analyse the outputs of all modulated signals.
\item Try the phase discrimination method for SSB signal generation.
\end{enumerate}
