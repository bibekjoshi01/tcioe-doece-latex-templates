\subsection{Scope and Application}


\subsubsection{Project Capabilities}

The project is capable of the following:
\begin{itemize}
    \item Real-time measurement of CO₂, PM2.5, VOC, temperature, and humidity
    \item Data logging and visualization on mobile or web
    \item Alert system for threshold breaches (SMS, email, or app notifications)
    \item Automated control of fans, purifiers, or ventilation devices
    \item Modular design for easy expansion with additional sensors
\end{itemize}


\subsubsection{Project Limitations}

The scope of this project is subject to the following limitations:
\begin{itemize}
    \item Accuracy of measurements depends on sensor quality and calibration; readings may be affected by temperature, humidity, or sensor drift
    \item Automated control is limited to compatible devices (e.g., fans, purifiers) and cannot address structural ventilation issues in buildings
    \item Designed primarily for indoor environments; may not scale effectively for large halls or industrial spaces without significant modifications
    \item Real-time monitoring and IoT features require stable internet connectivity; performance may degrade in areas with poor network access
\end{itemize}


\subsubsection{Project Applications}

The proposed system can be applied in the following domains:
\begin{itemize}
    \item Homes, offices, classrooms, and laboratories
    \item Can integrate with HVAC systems or standalone purifiers
    \item Real-time mobile or web dashboard for monitoring
\end{itemize}
