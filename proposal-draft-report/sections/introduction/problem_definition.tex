\subsection{Problem Definition}

Indoor air quality is often overlooked despite its critical impact on health. 
Current monitoring solutions are either costly or non-interactive. There is a 
need for a cost-effective, automated, and real-time IAQ monitoring and control 
system that can alert users and actuate corrective measures when pollutant levels 
exceed safe thresholds.
