# IOE, Thapathali Campus (DOECE) LaTeX Templates

Clean, ready-to-use LaTeX templates for IOE Thapathali Campus (DOECE) reports.

Designed for:

- Lab Reports
- Minor Projects
- Major Projects
- Assignments

## 📁 Available Templates

1. Lab Report: `lab-report/`
2. Proposal Draft Report: `proposal-draft-report/`
3. Proposal Report: `proposal-report/`
4. Progress Report: `progress-report/`
5. Final Report: `final-report/`

## ⚙️ Requirements

- Works directly on Overleaf or Any other Online Latex Processors
- Local Setup

## ❗ Rules

- Do NOT change formatting unless required
- Only edit content inside sections
- Keep structure intact for consistency

## 🤝 Contribution

If you find formatting issues or want improvements, open an issue or PR.
